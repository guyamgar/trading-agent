# Guy Trade - Multi-Agent BTC Trading System

מערכת מסחר רב-סוכנית ל-BTC/USDT עם 11 סוכני AI שלומדים, מתפתחים, ומקבלים החלטות מסחר.

## הסוכנים

1. **הצייד** - סורק setups בשוק
2. **המנתח הטכני** - מנתח אינדיקטורים
3. **מנהל הסיכונים** - בודק R:R ועמלות
4. **קורא ההקשר** - מצב מקרו של השוק
5. **החוקר ההיסטורי** - מחפש דפוסים דומים בעבר
6. **ראש הצוות** - מסנתז ומחליט
7. **פרקליט השטן** - מערער על דחיות לא מוצדקות
8. **המבקר** - בודק אם הניתוח לפני העסקה היה נכון
9. **המאמן** - מפיק לקחים מכל עסקה
10. **הסטטיסטיקאי** - מעדכן מדדים
11. **המכוון** - מנתח את כל המערכת ומציע שיפורים
12. **השופט** - מבקר על המלצות המכוון (חיצוני)

## שלבים

- **שלב 1 (פייפר):** $1K → קריטריונים סטטיסטיים (PF>1.5, WR>50%, 30+ trades)
- **שלב 2 (paper-live):** דאטה חי, shadow trading, 30 trades נוספים
- **שלב 3 (real money):** מסחר אמיתי בבייננס

## הפעלה

```bash
python3 bot.py
```

הבוט פעיל עם 3 threads רקע:
- 🤖 Auto-scanner (כל 15 דק')
- 🔴 Live monitor (כל 5 דק')
- 🌙 Daily summary (23:00 ישראל)

## ארכיטקטורה

```
agents/        - הסוכנים והפרומפטים
data/          - חיבור Binance + אינדיקטורים
scripts/       - learn_daily, test scripts
memory/        - זיכרון מצטבר (trades, lessons, account)
bot.py         - שליטה מטלגרם
live_check.py  - בדיקה חיה
live_monitor.py - מעקב אחרי המלצות פתוחות
auto_apply.py  - יישום אוטו של המלצות המכוון
```
